# 画像仕分けツール

Google Drive内の画像ファイルをブラウザで確認し、チェックボックスで選択して仕分けるツールです。

## 機能

- 画像をサムネイル表示（ファイル名付き）
- チェックボックスで画像を選択
- チェックした画像 → `2_仕分け後元画像/日付フォルダ` に移動
- チェックしなかった画像 → `4_予備保管ファイル` に移動

## 使い方

1. ターミナルで以下のコマンドを実行：

```bash
cd "/Users/johnstakickm1pro/claude/ローカル仕分け"
python3 image_sorter.py
```

2. ブラウザで `http://localhost:5000` を開く

3. 画像を確認してチェックボックスで選択

4. 「移動実行」ボタンをクリック

## 操作ボタン

- **すべて選択**: すべての画像にチェック
- **すべて解除**: すべてのチェックを外す
- **移動実行**: 選択状態に応じて画像を移動

## 移動先

- チェックあり: `/テスト環境/2_仕分け後元画像/YYYYMMDD/`
- チェックなし: `/テスト環境/4_予備保管ファイル/`
