# Changelog

## [Unreleased]

### Added
- 同日に複数回実行した場合、枝番付きフォルダを自動作成する機能を追加（例：20251014_01, 20251014_02）
- 選択された画像を枝番付き日付フォルダに移動するように改善
- フィルタ間で画像の選択状態を維持する機能を追加
- 全フィルタの選択数を合計表示する機能を追加
- 終了ボタンでブラウザタブを自動クローズする機能を追加
- 画像読み込み失敗時の自動リトライ機能を追加（最大3回）
- ブラウザネイティブの遅延読み込み（loading="lazy"）を実装
- 画像読み込み完了時のフェードイン効果を追加

### Changed
- 日付フォルダ命名規則を `YYYYMMDD` から `YYYYMMDD_NN` 形式に変更
- 選択状態をlocalStorageで管理し、フィルタ切り替え時に保持されるように改善
- 移動実行時に全フィルタの選択状態を反映するように改善

### Fixed
- フィルタを切り替えると選択状態が失われる問題を修正
- 選択数のカウントがリアルタイムで更新されない問題を修正
- 移動実行時に現在のフィルタの画像のみが対象になる問題を修正
- ネットワーク遅延による画像が表示されない問題を改善（自動リトライ機能で対応）
- 画像を選択していないのに選択中画像の数が計上される問題を修正（localStorageのクリーンアップ機能を追加）
